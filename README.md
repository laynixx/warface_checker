# Warface Accounts Checker

Многопоточный чекер аккаунтов с модулем автоматического обхода защиты на базе PyQt5. Скрипт проверяет базу аккаунтов через прокси, авторизуется и парсит данные об аккаунте.

## Anti-Bot Bypass

Для получения доступа к статистике софт автоматически решает динамическую JS-защиту (`n.js`), генерируя валид куки `n_js_t` и `n_js_d`:
1. **JSFuck Деобфускация:** Извлечение динамических переменных и их математическое вычисление в изолированном контексте через `js2py`.
2. **Стеганография:** Попиксельный анализ первых 32-х пикселей HTML5 Canvas-изображения через `Pillow`. Скрипт извлекает скрытый в цветовых каналах (RGB) бинарный ключ, замаскированный под картинку.
3. **Крипто-алгоритм `asm_a`:** Математическое вычисление хэша с использованием низкоуровневых типов данных (`ctypes.c_int32`) и побитовых операций (XOR, битовые сдвиги).

## Функционал

* **Защита от блокировок:** Поддержка прокси-серверов (HTTP/HTTPS) со случайной ротацией для каждого потока.
* **Интерфейс:** Табличное отображение результатов и логгер работы в реальном времени.
* **Проверка прокси перед работой:** Отправка запроса на сайт перед началом работы
* **Парсинг статистики:** Извлечение точных данных о количестве совершенного доната и параметров профиля, доступ к которым открывается только после успешного прохождения антибот-фильтров.


## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com
cd warface-pyqt5-checker
```

2. Установите необходимые библиотеки:
```bash
pip install -r requirements.txt
```

3. Запустите чекер:
```bash
python main.py
```

## Зависимости
* `PyQt5` - построение пользовательского интерфейса.
* `requests` - работа с сетевыми запросами и сессиями.
* `js2py` - выполнение JSFuck скриптов для получения динамических переменных защиты.
* `Pillow` - попиксельный анализ Canvas-картинок.
