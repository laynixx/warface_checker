# Warface Accounts Checker & Parser (PyQt5)

Многопоточный чекер аккаунтов с модулем автоматического обхода защиты n_j_d куки (JSFuck + HTML5 Canvas) на базе PyQt5. Скрипт проверяет базу аккаунтов через прокси, авторизуется и парсит количество доната со страницы профиля.

## Функционал

* **Многопоточность
* **Защита от блокировок:** Поддержка прокси-серверов (HTTP/HTTPS) со случайной ротацией для каждого потока.
* **Warface Bypass Engine:** Встроенный парсер JSFuck-выражений через `js2py` и декодер Canvas-изображений для генерации валидных куки (`n_js_t`, `n_js_d`).
* **Интерфейс:** Табличное отображение результатов (Email, Статус, Донат) и логгер работы в реальном времени.
* **Проверка прокси перед работой** Отправка запроса на https://warface.ru перед началом работы

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com
cd warface-pyqt5-checker
```

2. Установите необходимые библиотеки:
```bash
pip install -r requirements.txt
```

3. Запустите чекер:
```bash
python main.py
```

## Зависимости
* `requests` — работа с сетевыми запросами и сессиями.
* `js2py` — выполнение JSFuck скриптов для получения динамических переменных защиты.
* `Pillow` — попиксельный анализ Canvas-картинок.
